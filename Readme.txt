■概要
JR東日本E233系電車などのドア上に設置されているLCD式車内案内表示装置をPC上で再現するソフトウェアです。
路線・列車などの情報はXMLファイルに記述され、参照するファイルを切り替えることで表示する路線・列車を
変更可能です。また、XAMLで記述されたテーマファイルを切り替えることで様々な車両の表示装置を再現可能です。
多数のウィンドウの一括管理が可能で、複数列車の同時表示や、同一列車の左右・号車別の表示が行えます。

■インストール方法
アーカイブを任意の場所に展開してください。

■アンインストール方法
展開されたファイルをすべて削除してください。本ソフトウェアはレジストリを使用しません。

■主な操作
管理ウィンドウ > 列車設定リスト
- Space	ウィンドウリストに追加 <左ダブルクリック>でも操作可能

管理ウィンドウ > ウィンドウリスト
- Space	実行 [F5] <左ダブルクリック>でも操作可能
- Del	ウィンドウを閉じる / リストから削除 <右ダブルクリック>でも操作可能

表示装置ウィンドウ
- Esc	ウィンドウを閉じる [Alt+F4]でも操作可能
- F5	実行/停止 [Space][Pause/Break]でも操作可能
- F6	始発駅から再開 [BackSpace]でも操作可能
- F7	自動切替の有効化/無効化
- F8	左右の表示切替
- F11	全画面表示
- →	次に移動 [↓][Enter]でも操作可能
- ←	前に移動 [↑][Shift+Enter]でも操作可能
- Num1	詳細路線表示
- Num2	全区間路線表示
- Num3	ドア開閉情報表示
- Num4	プラットホーム情報表示
- Num5	乗り換え路線表示
- Num6	警戒実施表示
- Num7	優先席案内表示
- Num8	携帯電話取扱表示

詳細については http://wataku-city.com/software/NextStation/reference/ を参照してください。
